Instrucciones Básicas para Programar en Python

1. Instalar Python

Primero, asegúrate de tener Python instalado. Puedes descargarlo desde el sitio oficial de Python:

Descargar Python

2. Escribir tu Primer Programa

Abre tu editor de texto preferido y escribe el siguiente código:

print("¡Hola, Mundo!")

Guarda el archivo con la extensión .py (por ejemplo, hola_mundo.py).

3. Ejecutar el Programa

En la terminal o línea de comandos, navega hasta la carpeta donde guardaste el archivo y escribe:

python hola_mundo.py

Deberías ver la salida:

¡Hola, Mundo!

4. Estructuras Condicionales

Aquí tienes un ejemplo simple de una estructura condicional en Python:

numero = 10
if numero > 5:
    print("El número es mayor que 5")
else:
    print("El número es menor o igual a 5")

Aprender Python es divertido y útil. ¡Sigue practicando!
